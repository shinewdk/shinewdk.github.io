## 1. npm install   
```bash
  npm install -g @sencha/ext-gen 
  ext-gen app -a
  cd ./my-app
  npm start
```    
폴더 생성 된다!! 샘플페이지도 잘 열린다   
   
## 2. sencha   
```bash
> sencha
```
